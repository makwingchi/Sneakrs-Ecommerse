# Sneakrs-Ecommerse

An E-commerse website with ***Get OTP (one-time password)***, ***Log in***, ***Register***, ***View product list***, ***View product details***, and ***Create product*** functions.

## Stack
- `HTML/CSS`
- `Bootstrap 4`
- `Javascript`
- `jQuery`
- `AJAX`
- `Spring Boot`
- `Spring MVC`
- `Apache Maven`
- `MyBatis`
- `MySQL`
